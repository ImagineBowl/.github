# ImagineBowl 🍚

ImagineBowl is an open-source software organization building modern applications and developer-focused tools.

## Projects

- **QuranDaily** - https://github.com/ImagineBowl/QuranDaily - QuranDaily is a native iOS app built with SwiftUI for reading and listening to the Quran daily.
- **Raven** - https://github.com/ImagineBowl/Raven - A local-first audiobook player for iOS.
- **EQfi** - https://github.com/ImagineBowl/EQfi - AI-powered system-wide equalizer for macOS
- **Sonance** – https://github.com/ImagineBowl/Sonance - A native iOS instrument tuner built with SwiftUI
- **AI Page Summarizer** – https://github.com/ImagineBowl/AI-Page-Summarizer - Summarize any webpage in one click — with the AI backend you choose.
- **VibeRite** – https://github.com/ImagineBowl/VibeRite - VibeRite is a native macOS writing assistant
- **AudioConverter** – https://github.com/ImagineBowl/AudioConverter - A full-featured audio conversion platform with both browser-based (WAV to MP3) and server-based (all formats) conversion capabilities.

- Lightweight open-source apps
